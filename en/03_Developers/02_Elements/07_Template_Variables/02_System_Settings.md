### Options that display system data

[^qt^] - time for database queries

[^q^] - database queries

[^p^] - time for PHP scripts to run

[^t^] - total time to generate the page

[^s^] - content source (base or cache)

[^m^] - amount of memory consumed

#### Example:
```	
Memory : [^m^], 
MySQL: [^qt^], [^q^] request(s), 
PHP: [^p^], 
Total time: [^t^], 
Document from [^s^]. 
```
